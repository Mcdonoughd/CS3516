To run the simulation, first open the command prompt and navigate to the proper directory, where project3.c and
student3.h are located. To begin either run the command: make


----

To run an instance of the client, run the following command:

.\project3 [options]

Options are a tracelevel and a fileinput for a given inital distance matrix and number of nodes.
If no file is given then it runs the default file provided.

If no options are given, the program will give the following prompts:

"Enter Trace Level:"

Given level 5 the program will output the content of SampleOutput.txt
